A software component to collate versions of a text, for instance in order to produce a critical apparatus.

CollateX resembles software used to compute differences between files (e.g. {@code diff}) or tools for sequence alignment, commonly used in Bioinformatics. While some of the techniques and algorithms are the same, this library aims for a flexible and configurable approach to the problem of finding commonalities and differences in texts, sometimes trading computational soundness or complexity for the user's ability to influence results. This is because the assessment of such findings in disciplines like Philology or more specifically in the field of Textual Criticism) is often based on hermeneutic inquiry and as such can be supported by computational means – but such findings are not necessarily computable.

CollateX allows to compare more than two text versions (in the following called "{@link eu.interedition.collatex.Witness witnesses}"). This achieved by merging the results of consecutive comparisons between a witness and all previously compared witnesses into a data structure called a {@link eu.interedition.collatex.VariantGraph variant graph}. @see collatex.net
